هذا المشروع هو تطبيق ويب تم تطويره باستخدام React وJavaScript، وفكرته الأساسية هي إنشاء واجهة متجر إلكتروني تعرض المنتجات بطريقة منظمة وسهلة للمستخدم. يتيح المشروع للمستخدم التنقل بين عدة صفحات مثل الصفحة الرئيسية، صفحة المنتجات، صفحة السلة، وصفحة تسجيل الدخول والتسجيل، بالإضافة إلى صفحة خاصة بالإدارة.

يعتمد المشروع على تقسيم العمل إلى مكونات وصفحات لسهولة التنظيم والتطوير، حيث تم إنشاء صفحات مستقلة مثل:
	•	الصفحة الرئيسية Home
	•	صفحة المنتجات Products
	•	صفحة السلة Cart
	•	صفحة تسجيل الدخول Login
	•	صفحة التسجيل Register
	•	صفحة الإدارة Admin
